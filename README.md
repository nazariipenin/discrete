# Topological Sort (DFS)

Генерація графів, візуалізація та аналіз часу

Python-реалізація топологічного сортування за допомогою DFS на
орієнтованих графах.\
Проєкт також містить генерацію випадкових графів, побудову матриці та
списку суміжності та порівняння їхньої швидкодії.

## Запустити онлайн

Можна легко протестувати у Google Colab (не потребує встановлення Python
та бібліотек).

## Вимоги

-   Python 3.10+
-   Matplotlib
-   NetworkX
-   Jupyter Notebook (опціонально)

## Структура роботи

### 1. Генерація випадкового орієнтованого графа

Модель Ерделша--Реньї: кожне можливе ребро (u → v) створюється з
імовірністю p.

### 2. Представлення графа у двох форматах

-   Матриця суміжності
-   Список суміжності

### 3. Топологічне сортування (DFS)

Алгоритм виконується на списку суміжності і повертає топологічний
порядок або виявляє цикл.

### 4. Візуалізація

-   Heatmap матриці
-   Відображення графа через NetworkX
-   Графік часу побудови

### 5. Аналіз продуктивності

Вимірюється час побудови: - матриці суміжності - списку суміжності

## Конфігурація

    n – кількість вершин
    p – імовірність ребра
    sizes – список розмірів для тестування
